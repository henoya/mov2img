# CLAUDE.md

このファイルは、このリポジトリでコードを扱う際のClaude Code (claude.ai/code) へのガイダンスを提供します。

## プロジェクト概要

画面録画から静止画像を抽出するBashスクリプトユーティリティです。このスクリプトは動画ファイル（MOV/MP4）を解析し、画面が静止している瞬間を検出して、それらのフレームをPNG画像として自動的に抽出します。操作マニュアルやドキュメント作成に便利です。

## 入出力言語設定

画面への出力、処理中、処理結果の表示の言語は、可能な限り日本語に翻訳すること。

個別対応が必要な場合
- `CLAUDE.md`
  - 英語で記述
  - 日本語に翻訳したファイルを`CLAUDE_jp.md`として作成
  - CLAUDE.mdに変更があった場合、変更内容を日本語に翻訳してCLAUDE_jp.mdに反映
- `CLAUDE_jp.md`
  - 日本語翻訳ファイルはユーザー参照用であり、実行のコンテキストでは使用されない
  - CLAUDE.mdに反映されていない追加変更がある場合、各部分の対応方法をユーザーに確認
    - 対応オプション
      - 英語に翻訳してCLAUDE.mdの対応部分を書き換えまたは追加
      - メモとしてそのまま残す
      - ユーザーに処理詳細を入力してもらい、処理を実行

## Markdownファイルのフォーマット

- テキスト内にファイル名、絶対パス、相対パスが記載される場合は、`CLAUDE_jp.md`のようにコード記法を使用すること。特にプロジェクト内のファイルパスの場合。

## よく使うコマンド

### スクリプトの実行
```bash
# 基本的な使い方 - 動画からフレームを抽出
./mov2img.sh -i recording.mov

# カスタムパラメータを指定
./mov2img.sh -i recording.mp4 -o frames -n screenshot -t 5 -f 15

# デバッグ用の詳細出力付き
./mov2img.sh -i tutorial.mov -t 2 -d 0.5 -v
```

### スクリプトパラメータ
- `-i, --input`: 入力動画ファイル（必須）
- `-o, --output`: 出力フォルダ名
- `-n, --name`: 抽出画像のベース名
- `-t, --threshold`: 差分閾値パーセント（デフォルト: 3%）
- `-f, --fps`: 処理フレームレート（デフォルト: 30）
- `-d, --duration`: 最小静止時間（秒）（デフォルト: 1.0）
- `-v, --verbose`: 詳細出力を有効化

## 依存関係

スクリプトに必要なもの：
- `ffmpeg` - 動画処理とフレーム抽出用
- `ffprobe` - 動画解析用
- `bc` - 浮動小数点計算用

macOSでのインストール：
```bash
brew install ffmpeg
```

## アーキテクチャ

スクリプトは以下の主要コンポーネントを持つモジュラーアーキテクチャに従っています：

1. **引数解析** (`parse_args`): コマンドライン引数を処理し、デフォルト値を設定
2. **環境検証** (`validate_environment`): 依存関係と入力ファイルの有効性をチェック
3. **フレーム抽出** (`extract_static_frames`): コアロジック：
   - ffmpegを使用して指定FPSでフレームを抽出
   - 連続フレームを比較して静止期間を検出
   - 各静止期間の中央フレームを出力画像として保存
4. **クリーンアップ**: スクリプト終了時に一時ファイルを自動削除

スクリプトはフレーム処理用の一時ディレクトリを使用し、trapベースのクリーンアップで適切なエラー処理を実装しています。
